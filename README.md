# IVKeyFetcher
A tool to fetch IVKey, generated with the help of ChatGPT.
## Usage
java -jar IvKeyFetcher.jar -e [version] [model]
java -jar IvKeyFetcher.jat -ivkey [file] [version] [model]
## Why I create this project?
Semaphorin cannot retrieve firmware IVKey due to limitations in the original FirmwareKeysDl.jar.
This project was created to solve this problem by generating a new tool with the assistance of ChatGPT.
